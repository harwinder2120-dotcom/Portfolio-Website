# JavaScript Projects

These are the JavaScript projects I created during my course. Each project helped me practice functions, DOM manipulation, event handling, and building interactive user interfaces.

---

## 📌 Projects

### 1. Pizza Menu
A menu application where users can select pizza size and toppings.  
The program calculates the total price based on the user's selections and displays the final order summary.

---

### 2. Tic‑Tac‑Toe Game
A classic two‑player Tic‑Tac‑Toe game.  
The game randomly selects which player goes first, allows players to choose their token, and highlights the winner with sound and visual effects.

---

### 3. Calculator
A simple four‑function calculator built with JavaScript.  
Users can add, subtract, multiply, and divide using a clean, responsive interface.
